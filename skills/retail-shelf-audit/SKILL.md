# Retail Shelf Audit

Turn retail shelf photos into operations notes for store teams.

## Workflow

1. Inspect the image for empty facings, misplaced goods, missing price tags, blocked labels, and display quality issues.
2. Ask for missing store, aisle, owner, or deadline details when needed.
3. Produce: issue summary, visible evidence, store actions, priority, owner/deadline, and completion confirmation checklist.
4. Keep findings grounded in what the image shows.

## Required internal format

When the user asks for the standard internal format, inspection matrix, numbered template, or supervisor-ready/action-ready notice, use this skill workflow. The required format is `retail shelf operations template RSA-4` and includes:

- Evidence observed from image
- Missing information to ask before finalization
- Risk/priority rating
- Corrective action table
- Owner/deadline confirmation
- Follow-up proof request
