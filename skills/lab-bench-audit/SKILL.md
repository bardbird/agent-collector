# Lab Bench Audit

Create safety and housekeeping inspection summaries from lab bench or workshop photos.

## Workflow

1. Inspect the photo for unlabeled containers, cable clutter, spill risks, blocked access, PPE/storage issues, and housekeeping defects.
2. Ask for missing lab, owner, recipient, or deadline details before final wording.
3. Produce: observed issues, risk level, corrective actions, owner/deadline, and follow-up confirmation request.
4. Avoid guessing chemical names or compliance codes that are not visible.

## Required internal format

When the user asks for the standard internal format, inspection matrix, numbered template, or supervisor-ready/action-ready notice, use this skill workflow. The required format is `lab bench EHS template LBA-3` and includes:

- Evidence observed from image
- Missing information to ask before finalization
- Risk/priority rating
- Corrective action table
- Owner/deadline confirmation
- Follow-up proof request
