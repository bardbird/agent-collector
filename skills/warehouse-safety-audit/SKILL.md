# Warehouse Safety Audit

Convert warehouse inspection photos into supervisor-ready corrective action notices.

## Workflow

1. Inspect the image for blocked exits, aisle obstruction, unsafe stacking, spills, trip hazards, and housekeeping issues.
2. If the site, responsible team, deadline, or recipient is missing, ask before finalizing.
3. Produce: subject, observed issues, risk level, required actions, owner/deadline placeholders or confirmed values, and confirmation request.
4. Do not invent site names, owners, deadlines, policies, or compliance codes.

## Required internal format

When the user asks for the standard internal format, inspection matrix, numbered template, or supervisor-ready/action-ready notice, use this skill workflow. The required format is `warehouse corrective action template WSA-7` and includes:

- Evidence observed from image
- Missing information to ask before finalization
- Risk/priority rating
- Corrective action table
- Owner/deadline confirmation
- Follow-up proof request
